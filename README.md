# 📈 Stock Price Prediction using Linear Regression

A Machine Learning project that predicts stock closing prices using **Linear Regression**. The project uses historical stock market data, performs preprocessing and feature engineering, trains a regression model, and evaluates its performance using standard regression metrics.

---

## 🚀 Features

* Download historical stock data using **Yahoo Finance**
* Data preprocessing and cleaning
* Feature engineering
* Train a **Linear Regression** model
* Predict future stock closing prices
* Model evaluation using:

  * Mean Absolute Error (MAE)
  * Mean Squared Error (MSE)
  * Root Mean Squared Error (RMSE)
  * R² Score
* Visualize Actual vs Predicted stock prices

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* yFinance

---

## 📊 Model Evaluation

The model performance is measured using:

* MAE (Mean Absolute Error)
* MSE (Mean Squared Error)
* RMSE (Root Mean Squared Error)
* R² Score

These metrics help evaluate how accurately the model predicts stock prices.

## 📈 Output

The project generates:

* Predicted stock prices
* Actual vs Predicted comparison graph
* Model evaluation metrics
